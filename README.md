# ЛАБОРАТОРНАЯ РАБОТА №4

**Тема:** RAII и управление ресурсами

---

## 1. Задание

Реализовать библиотеку `resource_core`, демонстрирующую корректное владение внешними ресурсами (например, файлами) через RAII и умные указатели.

---

## 2. Функциональные требования

### 2.1. Пространства имён

Вся библиотека размещается в пространстве имён `lab4::resource`.

### 2.2. Библиотека resource_core

Минимальные сущности:
- `ResourceError` — тип исключений для ошибок работы с ресурсами.
- `FileHandle` — RAII-обёртка над внешним ресурсом.
- `ResourceManager` — кеш, позволяющий получить ресурс и отслеживающий его актуальность.

### 2.3. Тесты

Добавьте юнит-тесты, проверяющие корректное владение, разделяемое использование ресурсов и очистку кеша.

---

## 3. Подсказки

- Разведите объявления и определения по `.hpp` и `.cpp`.
- Используйте правило нуля/пяти и гарантии безопасности исключений.
- Подумайте, как `std::shared_ptr` и `std::weak_ptr` помогают управлять временем жизни.
